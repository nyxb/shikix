# shikix-compat

compatible build of `shikix` to align with `shiki`.

> Work in progress. It's not 100% compatible yet, but feedback is welcome :)

## Install

```bash
npm i -D shikix-compat
```

Or set the alias

```json
{
  "dependencies": {
    "shiki": "npm:shikix-compat@0.0"
  }
}
```
